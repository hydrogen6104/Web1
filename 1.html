HTML이란 무엇인가?

Hypertext Markup Language(HTML) is
the standard markup language for
creating web pages
and web applications.

HTML elements are the basic units of HTML, and consist of tags that surround different types of content.

Web browsers do not display HTML tags, but use them to interpret the content of the page. Most HTML elements have an opening and closing tag, with content in between.
